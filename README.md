database-oriented:

🧃 [lawn](https://github.com/mentalblood0/lawn) is a key-value store with mutable data files without trees
🧶 [dream](https://github.com/mentalblood0/dream) is a tagged objects index and search engine with advanced algorithm backed with lawn
👝 [trove](https://github.com/mentalblood0/trove) is an arbitrary JSON-structured data, all-indexing storage backed with lawn and dream
🐈 [wool](https://github.com/mentalblood0/wool) is a theses storage backed with trove
🔮 [chant](https://github.com/mentalblood0/wool) is a telegram-based collaboration interface backed with wool

configuration-oriented:

🌌 [hiemal](https://github.com/mentalblood0/hiemal) is a statically typed functional programming language without own syntax
🪸 [cryama](https://github.com/mentalblood0/cryama) is a file-based interface to ollama
